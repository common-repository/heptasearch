HeptaSearch permet d'améliorer et de rajouter des fonctionnalités au moteur de recherche Wordpress livré par défaut.


== Description ==


Pour cette 1ère version, nous avons rajouté 2 grosses fonctionnalités :
 
1/ pouvoir accéder facilement aux historiques des recherches des internautes avec les résultats apportés. Cela permet au responsable du site de voir quels sujets intéressent ses utilisateurs et de concentrer ses contenus vers les besoins de ses lecteurs.

2/ pouvoir gérer/éditer une blacklist des mots inutiles ou de mots qui gêneraient la pertinence de la recherche. Cette blacklist est activable ou non.

D'autres fonctionnalités et réglages sont également présents comme vider automatiquement les logs des recherches après un certain laps de temps (au bout d'une heure, un jour, une semaine, un mois, un an ou jamais). 

Nous allons rajouter de nouvelles fonctionnalités très prochainement. 
Pour plus d'informations ou demandes d'évolution merci de nous contacter à hello@heptadeca.com

== Installation ==


1. Uploader les fichiers du plugin dans le répertoire `/wp-content/plugins/hepta_search`, ou installez-le directement à partir de la page 'Extensions' WordPress.
2. Activer le plugin à partir sur la page 'Extensions' de WordPress
3. Aller sur la page Réglages de HeptaSearch pour configurer le plugin
